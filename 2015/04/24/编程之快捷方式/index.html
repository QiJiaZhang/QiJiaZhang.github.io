工欲善其事,必先利其器

  • 开发技巧

单单使用Xcode，工作太单调，有很多插件可以让编程既有意思又高效起来。有了Xcode插件，编程也是如此美好。

终归一句话Lazy Code

http://alcatraz.io/打开这个之后，估计大家都会用了吧。再多啰嗦几句我必使用的插件吧。

FuzzyAutocompletePlugin

有了它之后，随便输入几个字母，相关的类、方法都显示出来了。

KSImageNamed

写完[NSImage imageNamed: 或者 [UIImage imageNamed:之后，图片通通显示出来。

ColorSense

同样写完[NSColor: 或者 [UIColor之后，要定义的颜色会显示出来。

VVDocumenter

上次介绍过的标注插件，真是不用就out了呀~

XAlign

让代码瞬间对齐，强迫者适用。

xcfui

可以找到没用的#import，代码提速有它的功劳。

VVDocumenter

XCode的快捷键们

介绍完插件之后，Xcode的快捷键也是开发中必须用起来的秘密武器。之前有个非常爱好iOS编程的伙伴告诉我说，没有快捷键我都不像编程了，那让我们看看快捷键能让编程变得多快捷吧。

Command 1~8

再Navigators中互相切换, Command 0 关闭navigation

Command Alt 1~6

在Inspectors互相切换,同理 Command Alt 0也时将它关闭掉

Control Command Alt 1~4

再Libraries中相互切换

Control 1~6

展示Jump Bar的下拉选项

Command + Enter:

显示standard editor

Command Alt Enter:

You can probably guess what this does. Indeed, It opens the assistant editor.

Command Alt Shift Enter:

打开revision control editor.

Command + Shift + Y.

打开或者关闭掉debugging area

还不知道这些快捷键的猿和媛们，快试试吧！或者你正在说这些快捷键太简单了（我听见你再说），好吧！下次我会分享一些Ninja Master才会的快捷操作，希望能对大家有用，期待下次的分享吧~
